# 🌿 Symbiotic Nest (共生之巢)

> *“这不仅是一栋房子，而是一个盛满童心、沉淀温柔的感官实验室。”*

## ✨ 愿景 (The Vision)

那是一栋带着浅米色石头外墙和赤陶色瓦片的法式双层 Maison，在和煦的阳光下散发着沉淀的温柔。面向 150 平大草地的那面墙被彻底改造成了一整排极简的黑框大落地窗，模糊了室内外的界限。

这里是孩子们探索世界的起点，也是大人们在繁忙生活中的精神避风港。

### ☀️ 夏 · 流动 (Summer)
巨大的老橡树撑开一把绿伞，阳光穿过摇曳的葡萄藤，洒在散落于草地上的 Stapelstein 翻转石水盆里，映出绿意与涟漪。

### 🍂 秋 · 折射 (Autumn)
一楼的大落地窗上，贴满了透明磁力片，拼成一棵巨大的“磁力许愿树”。夕阳那抹金红色的光穿透水晶折射面，把整个活动室的地板、墙壁、甚至原木色的长椅，都打上了红的、蓝的、明黄色的流光溢彩。孩子躺在地板上捕捉移动的光影，整个屋子像一个巨大的、会发光的万花筒，美得像个奇迹。

### ❄️ 冬 · 庇护 (Winter)
窗外是法国冬季标志性的淅淅沥沥小雨，冷风摇曳着光秃秃的枝桠。而窗内，地暖开得足足的，暖烘烘一片。
孩子们围坐在厚厚的地毯上，将 Kapla 原木积木叠向高处，试图去够窗上的雨滴。突然，高塔倾倒，发出清脆、治愈的“啪嗒啪嗒”声，像一场松木的交响乐。在原木香与孩童的咯咯笑声中，这里成了全世界最安全的避风港。

## 🧩 核心哲学 (Philosophy)

*   **去机构化 (De-institutionalization)：** 拒绝“中心”感，构建“家”的温暖与亲密。
*   **感官优先 (Sensory-First)：** 触觉、嗅觉、光影，是认知的基石。
*   **低保真，高想象 (Low-Fi, High-Imagination)：** 用原木、泥土、原石替代塑料，让环境本身成为玩具。
*   **共生空间 (Symbiotic Space)：** 孩子们在 L1 释放纯真，大人们在 L2 享受静谧，通过“治愈白噪音”达成和谐共生。

## 🗺️ 空间映射 (Mapping)

*   **室内 (Indoor):** 2层，80-100 $\text{m}^2$ (橡木地板 / Espresso 咖啡香 / 万花筒光影)。
*   **室外 (Outdoor):** 150 $\text{m}^2$ (老橡树 / 葡萄架 / 泥巴厨房 / 赤足探索小径)。

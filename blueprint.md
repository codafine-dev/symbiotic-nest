# Symbiotic Nest (共生之巢 / Le Nid Symbiotique) 空间蓝图 v1.2

## 1. 核心哲学 / Core Philosophy / Philosophie
* **去机构化 / De-institutionalization / Dé-institutionnalisation：** 拒绝“中心/学校”感，构建“Maison (家)”的温暖与亲密。
* **感官优先 / Sensory-First / Priorité aux Sens：** 认为触觉、嗅觉和身体经验是认知发育的基石。
* **低保真，高想象 / Low-Fi, High-Imagination / Basse fidélité, Haute imagination：** 用自然、原始的材料替代标准化的塑料教具。
* **小而美 / Lean & Beautiful / Petit et Beau：** 极简的规模，极高的单点质量。

## 2. 季节律动 / Seasonal Rhythms / Rythmes Saisonniers
空间不应是静止的，而应随着季节的更迭，与孩子们共同呼吸。

### ☀️ 夏 / Summer / Été (The Fluidity / La Fluidité)
* **视觉：** 巨大的老橡树撑开绿伞，葡萄藤摇曳。
* **触感：** 散落在草地上的 Stapelstein 翻转石水盆 $\rightarrow$ 捕捉绿意与涟漪。
* **氛围：** 轻盈、流动、被自然包裹的清凉。

### 🍂 秋 / Autumn / Automne (The Refraction / La Réfraction)
* **视觉：** 落地窗上的“磁力许愿树” (Connetix) $\rightarrow$ 夕阳金红色的光穿透水晶折射面。
* **体验：** 整个活动室化身为巨大的万花筒 $\rightarrow$ 孩子躺在地板上捕捉移动的光影。
* **氛围：** 流光溢彩、梦幻、与光共舞。

### ❄️ 冬 / Winter / Hiver (The Sanctuary / Le Sanctuaire)
* **对比：** 窗外淅淅沥沥的冷雨 vs 窗内暖烘烘的地暖。
* **声音：** Kapla 原木积木高塔倾倒时清脆、治愈的“啪嗒啪嗒”声 $\rightarrow$ 松木的交响乐。
* **嗅觉：** 浓郁的原木香 $\rightarrow$ 极致的安全感。
* **氛围：** 温暖、包裹、全世界最安全的避风港。

## 3. 视觉与感官调色盘 / Visual & Sensory Palette / Palette Visuelle et Sensorielle
### A. 外部美学 / Exterior / Extérieur
* **色调：** 浅米色石头外墙 + 赤陶色瓦片 (Terracotta)。
* **界面：** 极简黑框大落地窗 $\rightarrow$ 彻底模糊室内外界限。

### B. 室内氛围 / Interior / Intérieur
* **L1 (童心之境 / Realm of Childhood / Domaine de l'Enfance)：** 
    * **材质：** 暖色橡木地板 $\rightarrow$ 传递温润的触感。
    * **视觉锚点：** 低矮柜上的古风积木 $\rightarrow$ 营造“微型美育博物馆”的陈列感。
* **L2 (成人圣殿 / Adult Sanctuary / Sanctuaire des Adultes)：** 
    * **材质：** 静音地毯 $\rightarrow$ 木楼梯 $\rightarrow$ 高强度隔音板。
    * **气味：** 浓郁的 Espresso 咖啡香。
    * **听觉：** 孩童笑声 $\rightarrow$ 隔音过滤 $\rightarrow$ 治愈的白噪音。

## 4. 建筑规格 / Architectural Specs / Spécifications Architecturales
* **规模：** 2层，室内 80-100 $\text{m}^2$，室外 150 $\text{m}^2$。
* **功能分布：** 
    * **L1：** 社交、感官游戏、美育陈列、自然接口。
    * **L2：** 深度办公、父母休憩、精神避风港。
